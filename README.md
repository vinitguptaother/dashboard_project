# 📊 AI-Powered Stock Market Dashboard

This project is an interactive, real-time stock market dashboard powered by AI. It features intraday, swing, and long-term investment insights — using data from Trendlyne, Screener.in, Moneycontrol, TradingView, and live news sentiment.

---

## 🛠 Project Structure

```
dashboard_project/
├── backend/           # Node.js backend APIs (alerts, news, analytics)
├── .kiro/             # AI assistant interactions
├── home/              # Frontend layout and UI flow
├── install-mongodb.ps1 # Script to install MongoDB on Windows
```

---

## 🚀 Features

- 🧠 AI-based decision layers (Manus, Kiro)
- 🗺 Multi-layered market scanning (intraday to long-term)
- 📈 Real-time stock sentiment (news + charts)
- 📦 Integrated MongoDB backend
- 🔔 Alerts via WebSockets

---

## 🧪 How to Run Locally

1. **Install dependencies**:
   ```bash
   cd backend
   npm install
   ```

2. **Start MongoDB** (make sure it’s installed)

3. **Run backend server**:
   ```bash
   node test-server.js
   ```
